Daten an REST-API-Method mit HTTP-POST schicken



Die REST-API-Methode, die diesen POST-Request entgegennimmt, ist RestControllerFuerPostRequest.datenViaPostRequestEmpfangen(String) im Paket de.mide.restapidemo.rest3.

Wird z.B. "Lorem Ipsum" eingegeben, dann wird dieser Methode der folgende String übergeben: 

meinKey=Falsches+%C3%9Cben+von+Xylophonmusik+qu%C3%A4lt+jeden+gr%C3%B6%C3%9Feren+Zwerg.

Der Schlüssel meinKey wird mit dem Attribut name im HTML-Formular-Element textarea definiert. Sonderzeichen wie das Leerzeichen oder Umlaute werden hierbei durch das sog. URL Encoding ersetzt.
